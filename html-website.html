HTML Basics

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It defines the structure and layout of a web document by using various elements and tags.

HTML Elements:

  • <html> - The root element of an HTML page
  • <head> - Contains meta information about the document
  • <body> - Represents the content of the document
  • <h1> to <h6> - Heading elements
  • <p> - Paragraph element
  • <a> - Anchor element for creating links
  • <img> - Image element for embedding images
  • <ul> - Unordered list element
  • <li> - List item element

HTML Resources:
